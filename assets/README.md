# Image assets

## Naming format

- Input `<2_digits_increment_number>_<anyname>.png` (eg. 01_cat.png)
- Output `<2_digits_increment_number>_<knumber>_<anyname>.png` (eg. 01_k5_cat.png)

## Why PNG file?

It's lossless (no compression artifacts that would corrupt singular values), supports grayscale and RGB.
